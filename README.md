# Netflix Data Analysis Project

##  Overview
This project performs exploratory data analysis (EDA) on the Netflix Movies and TV Shows dataset to uncover trends, patterns, and insights related to content distribution.

## 🛠 Tools & Technologies
- Python
- Pandas
- Matplotlib

##  Analysis Performed
- Comparison of Movies vs TV Shows
- Country-wise content distribution
- Year-wise content trends
- Ratings analysis

##  Key Insights
- Movies dominate Netflix content compared to TV Shows  
- United States produces the highest number of titles  
- Significant increase in content production after 2015  
- Majority of content falls under TV-MA rating  

##  Conclusion
This project demonstrates how data analysis can be used to extract meaningful insights from real-world datasets and support data-driven decision making.
